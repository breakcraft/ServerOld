# Changelog

## 2025
- **May** - `README.md` updated to point to [LostCityRS/Server](https://github.com/LostCityRS/Server). This repository is now archived.
- Engine updated to new messages table schema.
- Numerous quest and cheat fixes, including Chompy Bird and other content patches.

## 2024
- Added new login server backend and account ban/mute logic.
- Many quest improvements and bug fixes across Mining Guild, Demon Slayer, and more.
- Launcher and website updates for world configuration and downloads.

## 2023
- Initial import of RuneScript and start of project history.
- Website and webclient introduced along with autogeneration of cert configs.
- Early engine expansions with teleport scripts, NPC queueing, and object property merges.
